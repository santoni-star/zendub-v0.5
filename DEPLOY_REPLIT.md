# Розгортання ZenDub на Replit 🚀

Цей посібник допоможе вам запустити сервер ZenDub на платформі Replit безкоштовно.

## 1. Створення Repl

1. Зайдіть на [Replit.com](https://replit.com/).
2. Натисніть **+ Create Repl**.
3. Натисніть **Import from GitHub**.
4. Вставте посилання на цей репозиторій:
   `https://github.com/santoni-star/zendub-v0.5`
5. Натисніть **Import**.

## 2. Налаштування та Запуск

Проект вже містить необхідні конфігураційні файли (`replit.nix` та `.replit`).

1. Після імпорту Replit автоматично встановить пакети (Node.js, FFmpeg).
2. Натисніть велику зелену кнопку **Run** зверху.
3. У консолі ви побачите:
   ```
   ZenDub Server v1.3 running on port 3000
   ```
4. У вікні "Webview" (справа або зверху) з'явиться URL вашого сервера. Він виглядає так:
   `https://zendub-v05.your-username.repl.co`

**⚠️ Скопіюйте цей URL.** Він знадобиться для налаштування клієнта.

## 3. Налаштування Клієнта (Userscript)

Тепер треба сказати скрипту в браузері, куди надсилати запити.

1. Відкрийте меню розширення **Tampermonkey** у браузері.
2. Знайдіть скрипт **ZenDub** і натисніть іконку "Редагувати" (олівець).
3. Знайдіть рядок (близько 35-го), де вказано `serverUrl`:
   ```javascript
   const CONFIG = {
       // serverUrl: 'http://localhost:3000', // Старий локальний
       serverUrl: 'https://zendub-v05.your-username.repl.co', // ВСТАВТЕ СЮДИ ВАШ REPLIT URL
       // ...
   };
   ```
   *(Переконайтесь, що не залишили зайвих слешів `/` в кінці URL).*

4. Натисніть `File -> Save` (або Ctrl+S).

## 4. Перевірка

1. Відкрийте YouTube відео.
2. Натисніть кнопку **DUB**.
3. Якщо все вірно, скрипт з'єднається з вашим Replit сервером, і почнеться озвучка.

---

### Важливі зауваження для Replit:
*   **Sleep Mode**: Безкоштовні Repl "засинають", якщо їх не використовувати деякий час. Перший запит після сну може зайняти 10-20 секунд (поки сервер прокинеться).
*   **Ресурси**: Replit має обмеження на CPU/RAM. Time-stretching (FFmpeg) працює, але може бути повільнішим, ніж на потужному ПК.
*   **HTTPS**: Replit автоматично дає HTTPS, тому в скрипті обов'язково використовуйте `https://`.
