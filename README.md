# ZenDub v0.5 🎧✨

**ZenDub** — це потужний інструмент для автоматичного дубляжу YouTube відео українською мовою в реальному часі. Використовує штучний інтелект для генерації голосу (TTS) та синхронізації з відео.

## 🌟 Основні можливості

*   **Якісний дубляж**: Використовує сучасні TTS двигуни (Microsoft Edge TTS) для природного звучання.
*   **Time-Stretching**: Автоматично підлаштовує швидкість озвучки під відео, зберігаючи висоту голосу (Native FFmpeg).
*   **Прогресивне відтворення**: Починає озвучку майже миттєво, завантажуючи решту у фоні.
*   **Розумна пауза**: Відновлює відтворення з правильного моменту після паузи.
*   **Вибір голосу**: Підтримка чоловічих та жіночих голосів.
*   **Кешування**: Зберігає згенеровані аудіофайли для миттєвого повторного відтворення.

## 🛠️ Встановлення

Система складається з двох частин: **Сервер** (для генерації аудіо) та **Клієнт** (скрипт для браузера).

### Крок 1: Запуск Сервера

Для роботи потрібен [Node.js](https://nodejs.org/) та [FFmpeg](https://ffmpeg.org/).

1.  Клонуйте репозиторій або завантажте код:
    ```bash
    git clone https://github.com/santoni-star/zendub-v0.5.git
    cd zendub-v0.5/server
    ```

2.  Встановіть залежності:
    ```bash
    npm install
    ```

3.  Запустіть сервер:
    ```bash
    node server.js
    ```
    Сервер запуститься на порту `3000`.

### Крок 2: Встановлення Клієнта

1.  Встановіть розширення для браузера **Tampermonkey** (або Violentmonkey).
    *   [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
    *   [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

2.  Створіть новий скрипт в Tampermonkey.
3.  Скопіюйте вміст файлу `client/zendub.user.js` у редактор скрипта.
4.  Збережіть (Ctrl+S).

## 🎮 Використання

1.  Відкрийте будь-яке відео на YouTube.
2.  Ви побачите панель управління ZenDub під відео (або кнопку "DUB" на панелі плеєра).
3.  Натисніть **"DUB"**, щоб увімкнути дубляж.
4.  Натисніть **"SUB"**, щоб увімкнути оригінальні субтитри (для перевірки).
5.  Використовуйте перемикачі для зміни голосу (Чоловічий/Жіночий).

## ⚙️ Технічні деталі

*   **API**: Використовує локальний сервер на Express.js.
*   **TTS**: Edge-TTS Universal.
*   **Audio Processing**: FFmpeg (фільтр `atempo`) для зміни тривалості без зміни тону.
*   **Синхронізація**: Клієнтський скрипт аналізує таймкоди субтитрів та надсилає бажану тривалість блоку на сервер.

## 📝 Вимоги

*   Node.js v14+
*   FFmpeg (має бути доступний у системному PATH)
*   Браузер Chrome або Firefox
